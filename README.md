# 3D Action Game

## 概要
Unityで開発した3Dアクションゲームです。
プレイヤーは敵モンスターと戦い討伐することで勝利します。

## 制作目的
もともとゲームが好きだったことをきっかけに「遊ぶ側から作る側になりたい」と思ったため、モンスターハンターを参考に制作しました。

## 主な機能
- プレイヤー移動
  前進 W 後退 S 左 A 右 D
- 通常攻撃
  左クリック
-回復
  R
-回避
  Space
-視点移動
 マウス
- HPシステム
- 敵AI（追跡・攻撃・待機）
- アニメーション制御
- ゲームクリア・ゲームオーバー判定

## 使用技術
- Unity
- C#
- NavMesh
- Animator

## 工夫した点
敵AIを状態ごとに管理し、
プレイヤーとの距離によって行動を切り替えるよう実装しました。
